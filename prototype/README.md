# Prototipos — IronLog

## Como ver los prototipos

1. Instalar dependencias: `pnpm install`
2. Ejecutar: `pnpm dev`
3. Abrir http://localhost:5173 en el navegador
4. Usar el selector de pantalla en la barra superior para navegar

## Notas de diseno

- **Dark-only** — No existe modo claro. Fondo #0A0A0A.
- **Vibe:** Energia y potencia. Numeros grandes en Geist Mono. Inspirado en Hevy.
- **Color tematico:** Slate/acero #64748B para decoracion. Brand-blue #3291FF para acciones.
- **Mobile-first:** Max width 430px, optimizado para uso con una mano.
- **Datos mock:** Todos los datos son ficticios para demostrar el layout.
